About This Application

Purpose

This Flask application was created as part of Lab 4.2.1 - Playing with HTML and CSS. The main goal is to demonstrate how to implement a responsive and user-friendly web interface using HTML, CSS, and the Bulma framework.

Features

Responsive Design

The application is fully responsive and works well on desktop, tablet, and mobile devices.

Improved Typography

We've improved typography with better font choices, sizes, and spacing for enhanced readability.

Visual Hierarchy

Clear visual hierarchy with color, size, and spacing makes it easier to navigate the content.

Data Visualization

Integration with Chart.js for creating interactive charts and graphs to visualize data.

Technologies Used

Flask Python HTML5 CSS3 JavaScript Bulma CSS Chart.js Font Awesome

About the Developer

Student at FIU

This application was developed as part of a coursework assignment at Florida International University. The goal was to demonstrate HTML and CSS skills in a Flask application context.

Feel free to explore the code and use it as a reference for your own projects!
